# Sistema de Gestión de Productos

Aplicación web desarrollada en HTML, CSS y JavaScript que permite gestionar productos, simular pagos y visualizar estadísticas desde un dashboard administrativo.

## Funcionalidades
- Gestión de productos
- Simulación de pagos
- Login de usuario
- Dashboard con estadísticas
- Pruebas automatizadas con Selenium

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Python + Selenium
