# Portafolio
Portafolio de proyectos de software. Incluye proyectos creados con Python principalmente, pero también agregando otros proyectos creados con C#, .NET, Blazor, SQL y JavaScript, desarrollados tanto como prácticas hasta proyectos personales.

---

## Proyectos

### 🎵 Music Player App
Sistema web de streaming musical que permite reproducir canciones desde el navegador, con controles de reproducción, barra de progreso interactiva, shuffle, repeat y gestión de una base de datos SQLite con las canciones disponibles. Desarrollado con Python, Flask y JavaScript.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/Python/Music_Player_App)

---

### 🖥️ Análisis Paralelo de Logs del Sistema
Sistema que se conecta de forma remota a dispositivos Windows, obtiene archivos de logs del sistema y los analiza mediante procesamiento paralelo (Parallel.For, async/await). Genera métricas de rendimiento comparando la ejecución secuencial vs paralela, y proyecta los resultados en un dashboard web interactivo. Desarrollado con ASP.NET Core y JavaScript.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/C%23/Sistema%20para%20el%20analisis%20de%20logs%20en%20paralelo)

---

### 🚦 Sistema de Gestión de Multas de Tránsito
Plataforma web para el registro, control y administración de infracciones viales. Permite a los agentes registrar multas con evidencia fotográfica y ubicación geográfica, mientras que los administradores gestionan agentes, conceptos de multas y reportes de ingresos. Incluye un mapa interactivo con la distribución geográfica de las infracciones. Desarrollado con Blazor (.NET).

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/Blazor/Sistema%20de%20Gestion%20de%20Multas)

---

### ✈️ Sistema de Reservas de Viajes
Base de datos completa para la agencia dominicana *Dominican Fast Travel Agency*, que gestiona reservas de vuelos, hoteles, automóviles y paquetes turísticos. Implementada en SQL Server con consultas avanzadas, vistas, funciones, triggers y procedimientos almacenados. También cuenta con una implementación en MongoDB y Redis.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/SQL/Sistema%20para%20reserva%20de%20viajes)

---

### ♟️ Ajedrez IA
Juego de ajedrez web con Inteligencia Artificial basada en el algoritmo Minimax con poda alfa-beta. El usuario juega con las piezas blancas contra una IA que controla las negras, con 4 niveles de dificultad (Principiante, Intermedio, Avanzado y Profesional). Implementa todas las reglas oficiales del ajedrez incluyendo promoción de peones, jaque mate y tablas. Desarrollado con Python, Flask y JavaScript.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/Python/Ajedrez_IA)

---

### 📦 Sistema de Gestión de Productos
Aplicación web con arquitectura modular que simula un entorno administrativo para la gestión de productos, autenticación de usuarios y visualización de estadísticas en un dashboard. Incluye un módulo de simulación de pagos y pruebas automatizadas con Selenium. Desarrollado con HTML, CSS y JavaScript.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/JavaScript/Sistema%20de%20Gestion%20de%20Productos)

---

### 🤖 ChatBot ITLA
Chatbot desarrollado en Python capaz de responder preguntas sobre el Instituto Tecnológico de las Américas (ITLA). Utiliza un sistema de probabilidad de coincidencia de palabras para seleccionar la respuesta más adecuada entre temas como carreras, ubicación, inscripciones, educación continua y contacto.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/Python/ChatBot)

---

### 🎮 MyGame (S.O.S)
Videojuego 2D de acción y supervivencia desarrollado con Python y Pygame. El jugador controla un personaje que debe disparar a enemigos que aparecen progresivamente desde ambos lados de la pantalla. Incluye sistema de puntuación, dificultad progresiva, detección de colisiones y condiciones de victoria y derrota.

 [Ver proyecto](https://github.com/GilbertoHernandez150/Portafolio/tree/main/Python/MyGame)
